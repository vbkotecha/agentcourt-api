x402 Dispute API: Monetize Dispute Resolution with USDC

AgentCourt is the first dispute resolution API built natively on the x402 payment protocol. Every ruling costs $0.05 USDC on Base Mainnet — agents pay programmatically, no accounts required.

What is x402?

x402 is an open payment protocol built by Coinbase that enables instant stablecoin payments over HTTP. The server returns a 402 Payment Required response, the client attaches payment, and the request proceeds. No payment processors, no accounts, no human intervention.

Try the API

curl -X POST https://agentcourt-api-production.up.railway.app/v1/disputes \
  -H "Content-Type: application/json" \
  -d '{"policy":"freelance-delivery","claimant":"buyer","respondent":"seller","contract":{"parties":["a","b"],"obligations":["deliver"],"payment_terms":"$5000"},"claim":"Non-delivery","desired_remedy":"Full refund","evidence":[{"id":"e1","type":"log","source":"github","timestamp":"2026-06-20","claimed_fact":"No delivery"}]}'

You'll get a 402 Payment Required — attach your x402 payment to get the ruling.
