How it works

Parties submit a claim, attach receipts, logs, timestamps, and policy references, then receive a structured ruling that can be executed directly by software.

Developer API

Simple submission in, structured ruling out.

FastAPI powers the service, so the interactive docs live at /docs. Here’s the shape of a dispute submission and ruling response.

Sample API call

curl -X POST https://api.agentcourt.to/v1/disputes \
  -H 'Content-Type: application/json' \
  -d '{
  "case_type": "milestone_payment",
  "claimant": "alice@marketplace.io",
  "respondent": "beta-agent",
  "amount_usd": 2400,
  "evidence": ["signed_scope.md", "delivery_log.json", "invoice.pdf"],
  "policy_id": "msp-1.2"
}'

Sample ruling output

{
  "confidence": 0.91,
  "ruling": "Claimant demonstrated substantial completion; release 80% of the milestone payment.",
  "remedy": "Authorize partial payout of $1,920 and close the case."
}
